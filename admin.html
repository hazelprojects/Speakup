🤖 Gemini Prompt — Paper JSON Banao
Yeh prompt copy karo → Gemini ko do + 2 paper images attach karo → JSON milega
Main tumhe 2 images de raha hun — ek Objective paper (MCQs) aur ek Subjective paper. Dono images ko dhyan se dekho aur bilkul exactly wahi questions likho jo images mein hain — ek lafz bhi change mat karna, koi apni taraf se mat likhna. Sirf sections array return karo — koi extra text nahi, koi class/year/board nahi, koi explanation/wajah nahi, sirf yeh exact format: [ { "type": "mcq", "title": "Section A — Objective (MCQs)", "instructions": "Yahan paper ki exact instructions likho", "questions": [ {"q": "Bilkul exactly wahi question jo paper mein hai", "opts": ["A","B","C","D"], "ans": 0} ] }, { "type": "qa", "title": "Section B — Short Questions", "instructions": "Yahan paper ki exact instructions likho", "questions": [ {"q": "Bilkul exactly wahi question jo paper mein hai", "ans": "", "hint": ""} ] }, { "type": "long", "title": "Section C — Long Questions", "instructions": "Yahan paper ki exact instructions likho", "questions": [ {"q": "Bilkul exactly wahi question jo paper mein hai", "ans": "", "hint": ""} ] } ] ZAROORI RULES: 1. Sirf sections ARRAY return karo — [ ] se shuru ho 2. MCQ mein ans 0-based index hoga (A=0, B=1, C=2, D=3) 3. Sirf wahi sections jo paper mein hain 4. Koi markdown, backticks ya extra text nahi 5. Questions bilkul word-for-word copy karo images se — koi change nahi 6. qa aur long mein ans aur hint bilkul khali rakho — "" — apni taraf se kuch mat likho 7. MCQ mein "exp" field bilkul mat likhna

Format: Rule Title | Formula | Example English | Example Urdu
Har line ek new story line — format: English line | Urdu line
